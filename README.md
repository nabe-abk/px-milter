# PX-Milter

It is a rule-based email filter using the "Sendmail Milter API".
You can create email filters with just a little perl code.

# Enviroment

  * Postfix or Sendmail
  * Perl 5.14 or later
  * [Sendmail::PMilter](https://github.com/avar/sendmail-pmilter)
    - Included in this repository under [Sendmail/](./Sendmail/). Sendmail::PMilter have [other license](./Sendmail/LICENSE).

----------------------------------------------------------------------
[Japanese]

このソフトは、Postfix または Sendmail（動作未確認）で動く、ルールベースSPAMフィルターの雛形です。
